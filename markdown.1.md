NAME
====

_printf - format and print data

SYNOPSIS
========

_printf FORMAT [ARGUMENT]...

DESCRIPTION
===========

Produces output according to a format.

# SUPPORTED FORMATS

%c    prints a single character

%s    prints a string

%%    prints a '%'
